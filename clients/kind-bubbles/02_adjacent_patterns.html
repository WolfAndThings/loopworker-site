Steal it · for Kind Bubbles
Run a monthly carousel: "5 things to do with your kid in [neighborhood] this [month] — the bubble-and-sensory edition." Mix Kind Bubbles event dates with adjacent free Bay Area things to do (museum free days, library kid hours, sensory-friendly venues). The non-Kind-Bubbles items make it useful enough to save; the Kind Bubbles dates are why it converts. One per month, evergreen format, Saturday morning post slot.

If it lands, it's the new launch template.

Expected return: 3–5 inbound posts to highly-targeted parent audiences. Phase 2 outreach lane.

The five-move starter list

If only five of these get tested, do these.

  1. Run one weekend round-up carousel a month — Pattern 01. @bayarea_toddlerexplorer hit 350 + 225 likes on the same format. You can localize to bubbles + sensory ideas.
  2. Use the date-bound launch template on every public event — Pattern 02. Pairs with your proven Comment Trigger.
  3. Pick one mom-blogger this month and offer a free experience for a review — Pattern 04. Five names exist in this scrape file alone; expand later.
  4. Post a 12-photo portfolio carousel after every booked event — Pattern 05. Predictable 50-80 like floor without trying for viral.
  5. Pair your highest-engagement reaction stills with one Chris-voice sentence — Pattern 06. Founder & Family voice on Reaction & Show photo = compound.
